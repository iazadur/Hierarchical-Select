## Features

- **Two UI Frameworks**: Supports both Ant Design and Shadcn UI out of the box
- **Cascading Selections**: Child dropdowns update based on parent selections
- **Asynchronous Loading**: Built-in loading states for data fetching
- **Multiple Selection**: Supports multi-select at any level
- **Error Handling**: Elegant error display and recovery
- **Fully Typed**: Complete TypeScript definitions
- **Responsive**: Works well on mobile and desktop
- **Accessible**: Built with accessibility in mind

## Installation

```bash
npm install @iazadur/hierarchical-select
```

Or with yarn:

```bash
yarn add @iazadur/hierarchical-select
```

## Basic Usage

```jsx
import HierarchicalSelect from '@iazadur/hierarchical-select';

function LocationSelector() {
  return (
    <HierarchicalSelect
      fields={[
        {
          index: 0,
          options: [
            { value: 'us', label: 'United States' },
            { value: 'ca', label: 'Canada' },
            { value: 'uk', label: 'United Kingdom' },
          ],
          label: 'Country',
        },
        {
          index: 1,
          options: [],
          label: 'Region',
          fetchOptions: async (parentValue) => {
            // Fetch regions based on selected country
            const response = await fetchRegions(parentValue);
            return response.data;
          }
        },
        {
          index: 2,
          options: [],
          label: 'City',
          multiple: true,
          fetchOptions: async (parentValue) => {
            // Fetch cities based on selected region
            const response = await fetchCities(parentValue);
            return response.data;
          }
        }
      ]}
      designSystem="antd" // or "shadcn"
      onChange={(values) => console.log('Selected:', values)}
    />
  );
}
```
